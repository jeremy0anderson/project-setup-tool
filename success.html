Project Setup Tool

Create a Trello board and a Github repo with the click of a button

Please note that, currently, the tool does not automatically create a Github repo or Trello board. We have not yet covered Node.Js, which is the method used by the Github API. Once we cover the concept, we will update the tool accordingly so that a Trello board and a Github Repo are created when the correspoding button is clicked. Thank you for your patience.
